# 数据大屏项目分析

## 项目概述

数据大屏项目是一个基于Vue 3 + TypeScript + Vite构建的现代化数据可视化平台，主要用于实验室检测数据的实时监控、统计分析和展示。项目包含多个功能模块，包括首页屏、数据屏、智慧驾驶舱和指挥中心大屏，为不同层级的用户提供直观的数据可视化界面。

## 项目结构

### 主要目录结构
- `src/` - 源代码目录
  - `api/` - API接口定义
  - `assets/` - 静态资源
  - `components/` - 组件库
  - `hooks/` - 自定义hooks
  - `router/` - 路由配置
  - `utils/` - 工具函数
  - `views/` - 页面视图
- `public/` - 公共资源
- `readme/` - 项目文档和截图

### 技术栈
- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **状态管理**：Vue 3 Composition API
- **路由**：Vue Router 4
- **UI组件**：Ant Design Vue
- **图表库**：ECharts 5
- **样式**：Less + Tailwind CSS
- **HTTP客户端**：Axios
- **工具库**：@vueuse/core

## 核心功能模块

### 1. 首页屏 (homeScreen)
- 当日委托数、待检任务数、当月已检任务数等关键指标展示
- 委托参数数量排行
- 最新消息轮播
- 委托数量趋势图
- 当日在检任务列表
- 功能室使用情况实时监控

### 2. 数据屏 (dataScreen)
- 任务统计分析
- 样品统计分析
- 检测统计分析
- 委托参数数量排行
- 视频监控集成
- 任务列表展示
- 人员统计分析
- 设备统计分析
- 功能室使用状态

### 3. 智慧驾驶舱 (smartCockpit)
- 公司概况
- 综合监管
- 母体概况
- 人力资源监管
- 物资设备监管
- 委托样品监管
- 环境监控监管
- 试验检测监管
- 工地试验监管
- 智能楼宇监控

### 4. 指挥中心大屏 (commandCenter)
- 首页概览
- 数据中心

## 技术特色

1. **响应式设计**：适配不同屏幕尺寸，支持宽屏显示
2. **实时数据更新**：定时刷新数据，确保数据实时性
3. **模块化组件**：封装可复用的组件，提高代码复用率
4. **数据可视化**：使用ECharts实现丰富的图表展示
5. **视频监控集成**：集成海康威视等视频监控系统
6. **单位定制化**：支持不同单位的定制化需求
7. **动画效果**：数字自增、轮播等动画效果提升用户体验

## 项目亮点

1. **多维度数据展示**：从不同角度展示实验室检测数据
2. **实时监控**：实时监控功能室使用状态和检测任务进度
3. **智能分析**：对检测数据进行统计分析，提供决策支持
4. **用户友好**：直观的界面设计，易于操作和理解
5. **可扩展性**：模块化设计，易于添加新功能和适配新需求

# 简历项目描述

## 1. 项目描述

数据大屏项目是一个基于Vue 3 + TypeScript构建的现代化数据可视化平台，为实验室检测数据提供实时监控、统计分析和可视化展示功能。项目包含首页屏、数据屏、智慧驾驶舱和指挥中心大屏等多个模块，支持多单位定制化需求，为不同层级用户提供直观的数据可视化界面。

## 2. 技术栈

- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **路由**：Vue Router 4
- **UI组件**：Ant Design Vue
- **图表库**：ECharts 5
- **样式**：Less + Tailwind CSS
- **HTTP客户端**：Axios
- **工具库**：@vueuse/core

## 3. 个人职责

- 负责项目前端架构设计和技术选型
- 实现首页屏、数据屏等核心模块的开发
- 封装可复用的组件，如数据统计卡片、图表组件等
- 集成ECharts实现数据可视化图表
- 实现响应式设计，适配不同屏幕尺寸
- 优化页面性能和用户体验
- 参与需求分析和功能设计

## 4. 技术亮点

- **模块化设计**：采用组件化开发方式，提高代码复用率和可维护性
- **实时数据更新**：实现定时数据刷新机制，确保数据实时性
- **数据可视化**：使用ECharts实现多种图表类型，包括折线图、柱状图等
- **响应式布局**：适配不同屏幕尺寸，支持宽屏显示
- **动画效果**：实现数字自增、轮播等动画效果，提升用户体验
- **单位定制化**：支持不同单位的定制化需求，通过URL参数动态加载配置

## 5. 核心功能模块

- **首页屏**：关键指标展示、委托参数排行、消息轮播、任务趋势图、在检任务列表、功能室使用情况
- **数据屏**：任务统计、样品统计、检测统计、视频监控、任务列表、人员统计、设备统计
- **智慧驾驶舱**：公司概况、综合监管、人力资源、物资设备、委托样品、环境监控、试验检测、工地试验、智能楼宇监控
- **指挥中心大屏**：首页概览、数据中心

该项目通过直观的数据可视化界面，帮助实验室管理人员实时掌握检测任务进度、设备使用状态和人员工作情况，为决策提供数据支持，提升实验室管理效率。
