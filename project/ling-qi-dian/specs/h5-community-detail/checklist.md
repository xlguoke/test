# h5-community-detail Checklist v5.0

---

## 帖子内容

- [ ] 头像、昵称、发布时间展示正确
- [ ] 文字内容完整显示
- [ ] 图片九宫格排版正确，点击可预览
- [ ] 路线信息卡片条件展示，数据正确
- [ ] 点赞数、评论数实时更新

## 点赞交互

- [ ] 点击心形图标切换点赞状态
- [ ] 动画效果同广场页（scale + 颜色）
- [ ] 点赞数即时 +1 / -1
- [ ] 接口失败不切换状态

## 删除帖子

- [ ] 仅作者本人可见删除按钮
- [ ] 管理员也可见删除按钮（若角色信息可达）
- [ ] 点击弹出确认弹窗："确定删除该帖子？"
- [ ] 确认后调用删除接口
- [ ] 删除成功 toast "已删除" 并返回上一页
- [ ] 删除失败 toast 错误

## 评论列表

- [ ] 一级评论展示：头像、昵称、内容、时间
- [ ] 二级回复缩进展示，带 @被回复者
- [ ] 时间显示相对时间（如 "2小时前"）
- [ ] 评论按时间升序排列
- [ ] 空评论显示占位文案

## 发表评论

- [ ] 底部输入栏固定，不随列表滚动
- [ ] 点击输入框键盘正常弹起
- [ ] 空内容时发送按钮 disable
- [ ] 发送中按钮显示 "发送中..." 并禁用
- [ ] 发送成功后清空输入、收起键盘、刷新列表
- [ ] 发送失败保留输入内容，toast 错误

## 边界情况

- [ ] 帖子不存在或已删除：显示空状态 + 返回按钮
- [ ] 网络异常 toast 提示
- [ ] 快速重复点击点赞/发送已防抖
