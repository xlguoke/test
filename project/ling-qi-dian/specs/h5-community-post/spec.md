# h5-community-post Spec v5.0

> 发帖页面前端规格
> 版本: v5.0 | 日期: 2026-05-23

---

## 1. 目标

提供用户发布社区帖子的页面，支持图文编辑、路线录入、发布。

---

## 2. 页面路由

`/pages/community/post`（或项目对应路由）

---

## 3. 数据流

- **图片上传**：调起微信选择图片 API → 用户选择 → 前端上传至云存储 → 获得 URL 存入表单
- **发布**：校验通过 → 调用 `community-post-create` → loading → 成功跳转详情页

---

## 4. 交互规格

### 4.1 内容输入

- 多行文本输入框，placeholder："分享你的骑行故事..."
- 自动增高或固定高度（依设计）

### 4.2 图片选择

- 点击 "+" 图标，调起 `wx.chooseImage` 或 `wx.chooseMedia`
- 限制最多 9 张
- 已选图片横向排列，每张右上角显示 "X" 删除按钮
- 删除后立即从列表移除，允许重新选择

### 4.3 路线录入

- 折叠/展开区域，标题："添加路线信息"
- 输入项：
  - 起点（文本）
  - 终点（文本）
  - 距离（数字，单位 km）
  - 时长（数字，单位 分钟）
- 所有路线字段可选，但至少填一项时四项需完整校验（或按产品需求放宽）

### 4.4 发布按钮

- 位置：底部固定
- 状态：
  - `content` 为空或仅空白字符 → **disable**，灰色背景
  - `content` 有有效内容 → **enable**，主题色背景
- 点击发布：
  - 按钮文案变为 "发布中..."，禁用点击
  - 调用 `community-post-create`
  - 成功：跳转帖子详情页 `/pages/community/detail?post_id=xxx`
  - 失败：toast 错误信息，按钮恢复 "发布"

---

## 5. 表单校验

| 字段 | 规则 |
|:---|:---|
| content | 去除首尾空白后长度 > 0 |
| images | 数组，长度 0~9 |
| route_info | 可选。若填写，start/end/distance/duration 均为必填；distance/duration 为数字 |

---

## 6. 错误处理

- 内容为空点击发布：按钮无响应（已 disable）
- 图片上传失败：toast "图片上传失败，请重试"
- 发布接口失败：toast 后端 message，按钮恢复可点
- 网络错误：toast "网络异常"

---

## 7. Done Contract

- [ ] 图片选择、展示、删除流程完整
- [ ] 路线录入区域可折叠，字段校验正确
- [ ] 发布按钮随内容状态 enable/disable
- [ ] 发布中 loading 状态正确，成功跳转详情页
