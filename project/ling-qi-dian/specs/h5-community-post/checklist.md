# h5-community-post Checklist v5.0

---

## 内容编辑

- [ ] 文本输入框正常输入、换行
- [ ] placeholder 显示正确

## 图片选择

- [ ] 点击 "+" 调起微信选择图片
- [ ] 最多选择 9 张，超出提示
- [ ] 已选图片按选择顺序横向排列
- [ ] 点击 "X" 删除对应图片，布局更新
- [ ] 删除后可继续选择补充

## 路线录入

- [ ] 折叠面板可展开/收起
- [ ] 起点、终点文本输入正常
- [ ] 距离、时长仅允许数字输入
- [ ] 表单校验通过后方可发布（若填写了路线）

## 发布按钮

- [ ] content 为空时按钮 disable，灰色
- [ ] 输入内容后按钮 enable，主题色
- [ ] 点击后按钮变为 "发布中..."，禁用二次点击
- [ ] 发布成功跳转帖子详情页，URL 带 post_id
- [ ] 发布失败 toast 错误，按钮恢复

## 边界情况

- [ ] 返回时若已编辑内容，弹出确认框
- [ ] 图片上传失败可重试
- [ ] 网络异常 toast 提示
- [ ] 发布过程中页面不可退出或给出提示
