# h5-community-index Checklist v5.0

---

## 布局与展示

- [ ] 瀑布流两列，列间距 8px，行间距 8px
- [ ] 图片保持原始宽高比，无拉伸变形
- [ ] 单张图片最大高度限制，避免过长
- [ ] 文字内容最多 2 行，超出显示 ...
- [ ] 头像、昵称、点赞数、评论数排版正确

## 点赞交互

- [ ] 点击心形图标触发点赞/取消点赞
- [ ] 动画：scale 1.0 → 1.3 → 1.0，时长 200ms
- [ ] 颜色：未点赞 #999，已点赞 #FF4D4F
- [ ] 接口失败时不播放动画，toast 提示错误
- [ ] 点赞数实时更新

## 图片预览

- [ ] 点击九宫格任意图片进入全屏预览
- [ ] 预览从点击的图片索引开始
- [ ] 支持左右滑动切换
- [ ] 支持关闭（点击/下滑）

## 刷新与加载

- [ ] 下拉显示 loading，松手刷新，列表重置
- [ ] 上拉到底部显示 "加载中..."
- [ ] 追加新数据到瀑布流
- [ ] 无更多数据时显示 "没有更多了"
- [ ] 快速滑动不触发多次加载请求（防抖/锁）

## 边界情况

- [ ] 空列表显示空状态
- [ ] 网络异常 toast 提示
- [ ] 点赞状态与后端保持一致（重新进入页面刷新）
