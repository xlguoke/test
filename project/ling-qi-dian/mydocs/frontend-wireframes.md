## 设计原则

### 1. 核心原则

| 原则 | 说明 | 应用 |
|:---|:---|:---|
| **简洁** | 减少视觉噪音，突出核心内容 | 大量留白、精简配色 |
| **直观** | 用户无需学习即可使用 | 符合心智模型的布局 |
| **反馈** | 每个操作都有明确反馈 | 加载状态、成功提示、错误提示 |
| **一致性** | 相同元素表现一致 | 统一组件、统一交互 |

### 2. 用户画像

**主要用户**: 25-40 岁机车爱好者
- 偏好硬朗、工业风格
- 注重效率和实用性
- 移动端使用为主

## 设计系统

### 1. 色彩系统

#### 主色调
```
Primary (品牌色):    #1A1A1A (深黑)     - 导航、重点按钮
Secondary:           #C9A962 (金色)     - 会员等级、高亮
Accent:              #FF6B35 (橙红)     - 强调、促销
```

#### 功能色
```
Success:             #10B981 (绿色)     - 成功状态
Warning:             #F59E0B (橙色)     - 警告状态
Error:               #EF4444 (红色)     - 错误状态
Info:                #3B82F6 (蓝色)     - 信息提示
```

#### 中性色
```
Text Primary:        #1F2937 (深灰)     - 主文本
Text Secondary:      #6B7280 (中灰)     - 次要文本
Text Tertiary:       #9CA3AF (浅灰)     - 辅助文本
Border:              #E5E7EB (边框灰)    - 边框、分割线
Background:          #F9FAFB (背景灰)    - 页面背景
Surface:             #FFFFFF (纯白)      - 卡片背景
```

### 2. 字体系统

#### 字体栈
```css
/* 中文 */
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif;

/* 数字/金额 */
font-family: "DIN Alternate", "Helvetica Neue", Arial, sans-serif;
```

#### 字号规范

| 级别 | 大小 | 字重 | 用途 |
|:---|:---:|:---:|:---|
| H1 | 24px | 600 | 页面标题 |
| H2 | 20px | 600 | 区块标题 |
| H3 | 18px | 500 | 卡片标题 |
| Body | 16px | 400 | 正文 |
| Small | 14px | 400 | 辅助文字 |
| Tiny | 12px | 400 | 标签、时间 |

### 3. 间距系统

```
基础单位: 4px

xs:   4px   (超小)
sm:   8px   (小)
md:   16px  (中)
lg:   24px  (大)
xl:   32px  (超大)
2xl:  48px  (巨大)
```

### 4. 圆角系统

```
none: 0px     (直角)
sm:   4px     (小圆角)
md:   8px     (中圆角) - 默认
lg:   12px    (大圆角)
full: 9999px  (全圆角) - 按钮、标签
```

### 5. 阴影系统

```css
/* 小阴影 - 卡片 */
shadow-sm: 0 1px 2px rgba(0,0,0,0.05);

/* 中阴影 - 悬浮卡片 */
shadow-md: 0 4px 6px -1px rgba(0,0,0,0.1);

/* 大阴影 - 弹窗 */
shadow-lg: 0 10px 15px -3px rgba(0,0,0,0.1);

/* 强调阴影 - 按钮点击 */
shadow-accent: 0 4px 14px rgba(201,169,98,0.4);
```

## 交互规范

### 1. 按钮状态

```
默认状态:   背景 #1A1A1A, 文字 #FFFFFF
悬停状态:   背景 #333333, 文字 #FFFFFF
按下状态:   背景 #000000, 文字 #FFFFFF
禁用状态:   背景 #E5E7EB, 文字 #9CA3AF
加载状态:   显示 Loading 图标
```

### 2. 表单验证

```
实时验证:   输入框失焦时验证
错误提示:   输入框下方红色文字提示
成功提示:   输入框右侧显示 ✓ 图标
提交验证:   点击提交时全量验证
```

### 3. 加载状态

```
页面加载:   骨架屏 (Skeleton)
列表加载:   底部 Loading 动画
操作加载:   按钮 Loading 状态
图片加载:   占位图 + 渐显动画
```

```
成功提示:   Toast 绿色背景，2秒自动消失

确认弹窗:   Modal 确认/取消按钮
加载提示:   Loading 遮罩，可取消
```

## 响应式设计

### 1. 断点定义

```
Mobile:     < 768px   (手机)
Tablet:     768px - 1024px  (平板)
Desktop:    > 1024px  (桌面)
```

### 2. 适配策略

**点单端/社区端 (Mobile First)**:
- 固定宽度 375px 设计稿
- 使用 vw/vh 适配不同屏幕
- 底部导航固定

**后台管理 (Desktop First)**:
- 最小宽度 1280px
- 侧边栏固定 200px
- 内容区自适应

### 3. 组件适配

| 组件 | Mobile | Tablet | Desktop |
|:---|:---|:---|:---|
| 商品列表 | 2列网格 | 3列网格 | 4列网格 |
| 侧边栏 | 隐藏 | 折叠 | 展开 |
| 表格 | 横向滚动 | 完整显示 | 完整显示 |
| 图表 | 简化 | 标准 | 完整 |
