# Checklist: H5 会员中心、权益、充值、积分、充值记录、用户资料

> 验收标准：每个检查项必须通过视觉检查、交互测试、代码审查三重确认

---

## 3.1 会员中心页

- [ ] 页面可访问，顶部渐变背景（主题色 → 主题色深）
- [ ] 头像 80px 圆形，白色边框，居中
- [ ] 昵称：font-size-lg, font-weight-bold, color-text-inverse
- [ ] 等级徽章：胶囊形，颜色对应等级（普通灰/青铜棕/白银银/黄金金/铂金白/钻石蓝/至尊紫）
- [ ] 资产卡片：白色卡片，margin-top: -40px，3 列布局
- [ ] 积分数值：font-size-2xl, font-weight-bold, color-primary
- [ ] 余额数值：font-size-2xl, font-weight-bold, color-secondary
- [ ] 累计充值数值：font-size-2xl, font-weight-bold, color-text-primary
- [ ] 升级进度条：背景 var(--color-border)，填充渐变主题色
- [ ] 提示文字："再充值 ¥XXX 即可升级为 XX 会员"
- [ ] 功能入口网格：3×3，图标 40px，背景 var(--color-primary-light)
- [ ] 会员码弹窗：底部滑入，二维码 200×200px，每 5 分钟刷新提示
- [ ] 会员码内容：LQMEMBER:{member_id}:{timestamp}，5 分钟有效期
- [ ] 点击遮罩或下滑关闭弹窗
- [ ] 离线处理：网络异常显示 "网络异常，无法生成会员码"
- [ ] 离线缓存码：显示上次缓存的会员码 + "离线码，可能已过期" 标注

## 3.2 会员权益页

- [ ] 当前等级权益卡片：等级图标 + 名称 + "当前等级" 标签
- [ ] 权益列表：饮品/工时费/配件折扣 + 机车特色权益
- [ ] 等级对比表：横向滚动，7 个等级
- [ ] 当前等级列高亮：背景 var(--color-primary-light)
- [ ] 表头固定（sticky）
- [ ] 升级提示：底部固定，"立即充值" 按钮

## 3.3 会员充值页

- [ ] 活动横幅：一期不显示（⏳ 二期上线）
- [ ] 充值档位卡片：未选中/选中/禁用三种状态
- [ ] 档位内容：充值金额 + 赠送积分 + 成长值 + 可达等级
- [ ] ¥2,980 档位："⭐ 推荐" 角标 + "成为青铜会员"
- [ ] 点击动画：scale 0.98→1.0
- [ ] 底部支付栏："实付 ¥XXX" + "立即充值" 按钮
- [ ] 未选档位时按钮禁用
- [ ] 充值成功页：显示充值金额、获得积分、获得成长值、当前等级
- [ ] 充值失败页："支付失败，请重试" + "重新支付" 按钮
- [ ] 活动结束过渡：二期实现

## 3.4 积分明细页

- [ ] 顶部卡片：渐变背景，当前积分 font-size-3xl

- [ ] 收入：color-success，支出：color-error
