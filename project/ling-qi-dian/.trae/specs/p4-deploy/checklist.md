# Checklist: 联调与部署

> 验收标准：每个检查项必须通过功能测试、性能测试、生产验证三重确认

---

## 6.1 Mock 切换

- [ ] 环境变量文件创建完成（.env.development / .env.production）
- [ ] VITE_USE_MOCK 开关生效
- [ ] 请求封装根据环境变量切换 API
- [ ] 全部 API 调用正常
- [ ] Mock 和真实 API 返回结构一致

## 6.2 端到端联调

- [ ] 微信授权登录链路完整
- [ ] Token 过期自动跳转登录
- [ ] 后台管理登录正常
- [ ] 点单支付链路完整（浏览→加购→下单→支付→回调→积分）
- [ ] 订单状态流转正确
- [ ] 积分发放准确
- [ ] 充值链路完整（选择档位→支付→回调→余额更新→等级检查）
- [ ] 积分抵扣计算正确
- [ ] 后台商品上架/下架 H5 同步
- [ ] 后台订单状态更新 H5 同步
- [ ] 后台积分调整会员实时更新

## 6.3 异常场景

- [ ] 支付超时：30 分钟未支付自动取消
- [ ] 支付超时：库存恢复
- [ ] Token 过期：自动跳转登录页
- [ ] Token 过期：登录后返回原页面
- [ ] 库存不足：正确提示，订单未创建
- [ ] 余额不足：正确提示，未扣款
- [ ] 网络异常：自动重试
- [ ] 网络异常：数据一致性

## 6.4 性能优化

- [ ] 首屏加载 < 3 秒
- [ ] 代码分割生效
- [ ] 图片懒加载生效
- [ ] 接口响应时间 < 500ms
- [ ] 数据库索引命中
- [ ] 打包体积优化（Tree Shaking + Gzip）

## 6.5 部署上线

- [ ] H5 端打包成功
- [ ] H5 端上传 uniCloud 前端托管
- [ ] Admin 端打包成功
- [ ] Admin 端上传 uniCloud 前端托管
- [ ] 自定义域名配置完成
- [ ] 公众号 JS 安全域名配置
- [ ] 授权回调域名配置
- [ ] 收钱吧回调 URL 配置
- [ ] 生产环境全流程走通
- [ ] 生产环境支付回调验证
