# Checklist: H5 商品详情、购物车、订单确认与支付

> 验收标准：每个检查项必须通过视觉检查、交互测试、代码审查三重确认

---

## 2.1 商品详情页

- [ ] 页面可访问，顶部图片区 280px，圆角 radius-xl（底部）
- [ ] 图片轮播支持左右滑动，指示器圆点同步
- [ ] 返回按钮：左上角，圆形 36px，背景 rgba(0,0,0,0.3)
- [ ] 商品名称：font-size-xl, font-weight-bold
- [ ] 评分：星星图标 + 分数
- [ ] 标签：主题色标签样式
- [ ] 价格：font-size-3xl, font-weight-bold, color-secondary
- [ ] SKU 选择：选中/未选中/禁用三种状态视觉正确
- [ ] 数量选择器：步进器 "-" / 数字 / "+"，最小 1，最大 99
- [ ] 数量为 1 时 "-" 禁用
- [ ] 底部操作栏：收藏 + 购物车 + 加入购物车 + 立即购买
- [ ] 点击加入购物车：toast "已加入购物车"
- [ ] 点击立即购买：跳转订单确认页

## 2.2 购物车页

- [ ] 页面标题 "购物车"
- [ ] 清空按钮：点击弹出确认弹窗
- [ ] 商品卡片：勾选框 + 图片 + 名称 + 规格 + 单价 + 数量调整
- [ ] 勾选框：圆形，选中背景 var(--color-primary)
- [ ] 数量调整："+" / "-" 按钮，最小 1 时 "-" 禁用
- [ ] 左滑删除：滑动 80px 显示红色删除按钮
- [ ] 失效商品：灰色遮罩 + "已下架" 标签，勾选框禁用
- [ ] 失效商品提示："X 件商品已失效，点击清理"
- [ ] 清理按钮：点击移除所有失效商品
- [ ] 底部结算栏：全选 + "已选 X 件" + 合计金额 + "去结算"
- [ ] 未选商品时 "去结算" 禁用
- [ ] 合计金额实时计算正确

## 2.3 订单确认页

- [ ] 桌号网格选择：3 列布局，常用桌号显示
- [ ] 选中桌号高亮（主题色边框 + 背景）
- [ ] "更多" 项：点击展开输入框
- [ ] 历史桌号优先显示
- [ ] 备注输入：多行文本，最大 200 字符，字数统计
- [ ] 商品清单：图片 + 名称 + 规格 + 数量×单价 + 分类标签
- [ ] 分类标签颜色：饮品(青绿)/工时费(橙)/配件(蓝)
- [ ] 会员折扣：可展开/折叠，显示饮品/工时费/配件折扣
- [ ] 普通用户显示 "充值成为会员享专属折扣"
- [ ] 积分抵扣：Switch 开关，开启后显示步进器
- [ ] 积分不足 100 时开关禁用，显示 "积分不足 100，无法使用积分抵扣"
- [ ] 非会员显示 "充值成为会员即可使用积分抵扣"
- [ ] 金额明细：商品总价 - 折扣 - 积分抵扣 = 实付金额
- [ ] 实付金额：font-size-2xl, font-weight-bold, color-secondary
- [ ] 提交订单：调用 order-create，成功跳转支付页

## 2.4 订单支付页

- [ ] 支付金额：font-size-4xl, font-weight-bold, color-secondary
- [ ] 订单号显示
- [ ] 支付方式：微信支付，选中/未选中状态
- [ ] 确认支付按钮：全宽，主题色背景
- [ ] 点击显示 "支付中..."
- [ ] 轮询查单：每 2 秒一次
- [ ] 支付成功：绿色对勾 + "支付成功" + 订单信息
- [ ] 支付失败：红色感叹号 + "支付失败" + 原因

## 2.5 订单列表页

- [ ] 页面标题 "我的订单"
- [ ] Tab 栏：全部/待取餐/已完成/已取消/已取餐
- [ ] 选中 Tab：color-primary + 底部主题色下划线
- [ ] 订单卡片：白色卡片，Logo + 订单号 + 状态标签
- [ ] 状态标签颜色：待取餐(橙)/已完成(绿)/已取消(红)
- [ ] 商品缩略图 + 名称 + 数量
- [ ] 实付金额：font-size-lg, font-weight-bold, color-secondary
- [ ] 取餐码：font-size-3xl, font-weight-bold, color-primary
- [ ] 取餐码仅"待取餐"和"制作中"状态显示
- [ ] 取餐码点击弹出放大弹窗，方便远距离扫码
- [ ] 取消订单确认弹窗："确认取消订单？" + "再想想"/"确认取消"
- [ ] 操作按钮：再来一单/评价
- [ ] 空状态："暂无订单"

## 2.6 订单详情页

- [ ] 状态时间轴：待付款 → 已支付 → 制作中 → 已完成
- [ ] 当前状态高亮，已完成显示对勾
- [ ] 每个节点显示具体时间
- [ ] 商品清单同订单确认页
- [ ] 金额明细同订单确认页
- [ ] 订单信息：订单号、桌号、备注、时间
- [ ] 操作按钮按状态显示正确
- [ ] "再来一单"：商品加入购物车，跳转购物车

## 2.7 主题变量检查

- [ ] 所有颜色引用 CSS Variable
- [ ] 所有字号引用 CSS Variable
- [ ] 所有间距引用 CSS Variable
- [ ] 所有圆角引用 CSS Variable
- [ ] 组件 style 中无任何硬编码值
