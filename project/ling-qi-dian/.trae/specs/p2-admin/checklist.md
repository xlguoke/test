# Checklist: 后台管理前端（vben-admin）

> 验收标准：每个检查项必须通过视觉检查、交互测试、代码审查三重确认

---

## 4.1 工程初始化

- [ ] vben-admin 框架配置正确，pnpm dev 启动成功
- [ ] 主题色配置为 #4ECDC4
- [ ] 路由配置完整：登录/Dashboard/商品/订单/会员/系统
- [ ] 菜单根据权限动态渲染
- [ ] super 角色显示全部菜单
- [ ] admin 角色隐藏账号管理菜单

## 4.2 登录页

- [ ] 左侧品牌图占 60% 宽度
- [ ] 右侧登录表单占 40% 宽度
- [ ] 标题 "欢迎回来"
- [ ] 账号输入框：placeholder、前缀图标
- [ ] 密码输入框：placeholder、前缀图标、眼睛切换
- [ ] "记住我" 复选框，默认勾选
- [ ] 登录按钮：loading 状态、禁用二次点击
- [ ] 登录成功：存储 Token，跳转 Dashboard
- [ ] 登录失败：显示错误信息
- [ ] 表单校验：账号非空、密码非空、密码长度 ≥6

## 4.3 Dashboard

- [ ] 4 个数据概览卡片：今日订单/营业额/新增会员/总会员
- [ ] 数值 font-size-3xl, font-weight-bold
- [ ] 变化百分比：绿色↑/红色↓
- [ ] 近 7 日营业额趋势图：ECharts 折线图
- [ ] 会员等级分布饼图：ECharts 饼图，7 个扇区
- [ ] 最近订单表格：最近 10 条
- [ ] 快捷入口：4 个按钮

## 4.4 商品管理

- [ ] 商品列表页：搜索表单 + 表格
- [ ] 搜索字段：名称、分类、状态
- [ ] 表格列：图片、名称、分类、价格、库存、状态、标签、操作
- [ ] 库存 <10 红色高亮
- [ ] 状态 Switch 开关
- [ ] 分页排序正常
- [ ] 商品编辑页：左侧表单 + 右侧预览
- [ ] 表单字段：名称、分类、描述、图片上传、标签、状态
- [ ] SKU 管理：动态表格，添加/删除行
- [ ] 表单校验：至少 1 个 SKU、价格 >0
- [ ] 预览卡片实时更新

## 4.5 订单管理

- [ ] 订单列表页：搜索表单 + 表格
- [ ] 搜索字段：订单号、状态、支付、时间、会员
- [ ] 表格列：订单号、会员、商品、金额、支付、状态、桌号、时间
- [ ] 状态标签颜色正确
- [ ] 分页排序正常
- [ ] 订单详情页：信息卡片 + 时间轴 + 商品 + 金额 + 会员
- [ ] 状态时间轴：当前状态高亮
- [ ] 操作按钮按状态显示正确

## 4.6 会员管理

- [ ] 会员列表页：搜索表单 + 表格
- [ ] 搜索字段：昵称、手机号、等级、注册时间
- [ ] 表格列：头像、昵称、等级、累计充值、成长值、积分、余额
- [ ] 等级 Tag 颜色对应
- [ ] 分页排序正常
- [ ] 会员详情页：信息卡片 + 积分调整 + 余额调整 + 充值记录 + 最近订单 + 最近积分
- [ ] 积分调整弹窗：调整值、原因、实时计算
- [ ] 余额调整弹窗：调整金额、原因、实时计算
- [ ] 充值记录列表：订单号、充值金额、赠送积分、成长值、状态、时间
- [ ] 支持负数调整

## 4.7 系统管理

- [ ] 账号列表：表格 + 新增弹窗
- [ ] 表格列：用户名、昵称、角色、状态、创建时间
- [ ] 操作：编辑/重置密码/禁用启用
- [ ] 新增账号弹窗：用户名、密码、昵称、角色
- [ ] admin 角色访问账号管理显示 403
- [ ] 个人设置：修改密码、昵称、退出登录

## 4.8 会员配置管理

- [ ] 等级权益配置页：表格 + 编辑弹窗
- [ ] 等级权益表格列完整：等级名称、升级门槛、折扣、积分倍率等
- [ ] 等级名称和升级门槛只读
- [ ] 编辑弹窗：InputNumber 组件，步长和范围校验
- [ ] 保存调用 `admin-member-levels-save` API
- [ ] 充值档位配置页：表格 + 编辑弹窗
- [ ] 充值档位表格列完整：档位ID、充值金额、赠送积分、成长值、可达等级、启用状态
- [ ] 启用状态 Switch 开关
- [ ] 编辑弹窗：1:1:1 模型默认值联动
- [ ] 保存调用 `admin-recharge-tiers-save` API

## 4.9 主题变量检查

- [ ] 所有颜色引用 CSS Variable 或 vben 主题变量
- [ ] 所有字号引用主题变量
