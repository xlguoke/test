# 功能室检测信息展示系统 / 功能室检测业务操作系统 / 多功能导览系统 / 设备小屏

## 项目介绍

授权设备屏系统，包含座式屏/设备屏/智慧屏配置大屏等

### 一、技术栈

vue3 + vant4 + vite4 + pinia + tailwindcss + amap(高德地图) + echarts + threejs

### 二、详细

目前通过智慧屏授权绑定的设备屏项目基本都在这个项目内，通过路由控制不同的屏
截至当前（2025-02-26），项目内共有13种屏：
- 坐式屏
- 设备小屏
- 功能室大屏
- 功能室门口屏
- 标准物质间
- 化学药品室
- 标准养护室
- 样品室
- 留样室
- 外检室
- 外检室数据大屏
- 室内屏
- 指挥中心驾驶舱（这个屏是特殊的，通过 http://192.168.2.65/-/facility-inspection-system/index.html?unitCode=gfzx#/commandCenterScreen 访问）

## 访问方式
### 一、预览
授权设备屏通过在 ``ilis->海特后台管理->数据屏->智慧屏配置`` 中进行屏配置后可点击预览查看
### 二、配置实施
1. 配置屏幕 在 ``ilis->海特后台管理->数据屏->智慧屏配置`` 中进行屏配置
2. 在设备终端打开 http://192.168.2.65/-/facility-inspection-system/ 进入绑定页
3. 复制绑定页``大屏编号``,配置到第一步中的``屏幕编号``字段中保存
4. 在绑定页输入单位编码，点击我已绑定即可
