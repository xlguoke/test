# 重庆市公路水运工程质量检测管理系统

## 项目概述

重庆市公路水运工程质量检测管理系统是一个专门为重庆市公路水运工程质量检测行业设计的综合管理平台。该系统旨在规范检测机构的检测行为，提高检测数据的准确性和可靠性，加强行业监管，保障公路水运工程质量。

## 系统架构

### 技术栈

#### 后端
- Java 11+
- Spring Boot 2.7.7
- MyBatis-Plus
- PostgreSQL 数据库
- Redis 缓存
- Minio 对象存储
- Activiti 工作流引擎
- Docker 容器化部署

#### 前端
- Vue 3
- TypeScript
- Ant Design Vue
- ECharts 数据可视化
- Axios 网络请求
- Vite 构建工具

### 系统模块

#### 后端模块
1. **核心业务模块**

   - 人员管理
   - 新闻管理

2. **系统功能模块**
   - 权限管理

   - 微信公众号集成
   - 数据同步
   - 定时任务

#### 前端模块

   - 首页展示

   - 机构查询
   - 检测业绩查询

2. **后台管理 (web)**
   - 系统管理

## 系统特点

1. **安全性**：集成了Spring Security，实现了细粒度的权限控制
2. **可靠性**：使用PostgreSQL数据库，保证数据的一致性和可靠性
3. **可扩展性**：模块化设计，便于功能扩展
4. **易用性**：前端使用Ant Design Vue，界面美观，操作便捷
5. **实时性**：集成了Redis缓存，提高系统响应速度
6. **可部署性**：支持Docker容器化部署，便于系统维护和升级

## 部署方式

- 支持Docker容器化部署
- 支持多环境配置（开发、测试、生产）
- 支持子路径部署

## 项目价值

1. **规范行业行为**：通过系统管理，规范检测机构的检测行为，提高检测数据的准确性和可靠性
2. **提高监管效率**：实现了检测数据的实时采集和分析，便于行业监管
3. **提升服务质量**：通过门户网站，为公众提供便捷的信息查询服务
4. **促进行业发展**：通过数据统计和分析，为行业发展提供决策支持

## 简历项目描述

### 项目描述
重庆市公路水运工程质量检测管理系统是一个综合管理平台，旨在规范检测机构行为，提高检测数据准确性，加强行业监管，保障公路水运工程质量。系统分为门户网站和后台管理两部分，集成了检测业绩管理、机构管理、新闻管理等核心功能。

### 技术栈
- **后端**：Java 11+, Spring Boot 2.7.7, MyBatis-Plus, PostgreSQL, Redis, Minio, Activiti
- **前端**：Vue 3, TypeScript, Ant Design Vue, ECharts, Axios, Vite
- **部署**：Docker容器化
