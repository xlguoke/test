# 试验检测管理信息系统 - App端

## 项目概述

试验检测管理信息系统App端是一款专为试验检测现场工作设计的移动应用，作为PC端系统的延伸，为试验检测人员提供了便捷的移动工作方式。App端集成了二维码扫描、电子签名、蓝牙打印等移动端特有功能，大幅提高了试验检测工作的效率和灵活性。

## 技术栈

- **前端框架**：Vue 3 + TypeScript + Composition API
- **UI库**：Vant 4
- **样式**：TailwindCSS
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **构建工具**：Vite 6.2.4 + pnpm 10.7.1
- **网络请求**：Axios + 自定义API封装
- **移动端能力**：百度地图、二维码扫描、电子签名、蓝牙打印
- **其他工具**：qrcode、dayjs等

## 技术特点

- **组件化开发**：采用Vue 3的Composition API，实现组件的复用和逻辑的清晰分离
- **移动端适配**：使用Vant 4组件库，针对移动设备进行界面优化
- **响应式设计**：结合TailwindCSS，实现响应式布局，适配不同尺寸的移动设备
- **离线能力**：支持离线操作，网络恢复后自动同步数据
- **移动端特有功能**：集成二维码扫描、电子签名、蓝牙打印等移动端能力
- **跨平台支持**：支持微信小程序嵌入，实现多平台应用
- **性能优化**：通过代码分割、懒加载等技术，提高应用的加载速度和运行性能

## 核心技术实现

1. **移动端适配**：使用Vant 4组件库，针对移动设备进行界面优化
2. **扫码识别**：集成qrcode库，实现设备和样品的快速识别
3. **电子签名**：实现电子签名功能，支持报告和审批的签名确认
4. **蓝牙打印**：集成蓝牙打印功能，支持现场打印标签和报告
5. **离线存储**：使用localStorage和sessionStorage，支持离线操作
6. **状态管理**：使用Pinia管理全局状态，支持状态持久化
7. **网络请求**：统一的API封装，处理网络异常和重试机制
8. **地图集成**：集成百度地图API，支持地理位置相关功能
9. **微信小程序**：支持微信小程序嵌入，实现跨平台应用

## 核心功能模块

- **使用记录**：记录设备的使用情况和状态

### 样品管理

- **标准养护期**：管理样品的养护期
- **样品存储记录**：记录样品的存储情况

### 审核管理
- **移动端审批流程**：支持30+种业务类型的移动端审批
- **电子签名**：使用电子签名进行审批确认

### 温湿度监控
- **温湿度记录**：记录环境的温湿度数据
- **异常处置**：处理温湿度异常情况
- **巡查台账**：记录巡查情况

- **消息通知**：接收系统消息和通知
- **系统设置**：设置系统参数和偏好

## 项目架构优势

将移动端项目放置在PC端项目的子目录（/app/views/mobileApp）中具有以下优势：

1. **组件复用**：可以复用PC端的通用组件，如表单组件、弹窗组件等，减少重复开发
2. **工具类共享**：共享PC端的工具类库，如API封装、日期处理、文件操作等
3. **统一的技术栈**：使用相同的技术栈，便于团队协作和技术维护
4. **代码一致性**：保持代码风格和架构设计的一致性，提高代码质量
5. **便捷的同步更新**：当核心业务逻辑变更时，可以同时更新PC端和移动端
6. **统一的构建流程**：使用相同的构建工具和配置，简化构建和部署流程
