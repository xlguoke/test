# 海特博试云试验检测手簿软件项目分析

## 项目概述

海特博试云试验检测手簿软件是一款基于Vue 3和Cordova开发的平板应用，主要用于试验检测数据的采集、管理和上报。该软件集成了UDR（Universal Data Recorder）技术，支持在线和离线两种工作模式，能够满足不同场景下的试验检测需求。

## 技术栈

- **前端框架**：Vue 3 + TypeScript + Vite
- **状态管理**：Pinia
- **UI组件库**：Vant
- **路由管理**：Vue Router
- **移动平台**：Cordova（支持Android和浏览器）
- **数据存储**：SQLite
- **网络请求**：Axios
- **工具库**：dayjs, jwt-decode, zod

## 项目结构

项目采用了清晰的模块化结构，主要包括以下部分：

- **src/views**：包含主要页面组件，如登录、首页、任务管理、模板管理、我的设置等
- **src/components**：通用组件库
- **src/providers**：服务层，处理业务逻辑
- **src/stores**：Pinia状态管理
- **src/utils**：工具函数
- **src/type**：TypeScript类型定义
- **packages/custodian**：核心库，处理数据存储和业务逻辑
- **packages/migration**：数据迁移工具

## 核心功能

### 1. 试验任务管理
- 在线任务列表：显示服务器上的试验任务
- 离线任务列表：显示本地存储的试验任务
- 任务详情查看：查看任务的详细信息
- 任务状态管理：跟踪任务的执行状态

### 2. 数据采集与管理
- UDR表格：基于UDR技术的表格数据采集
- 设备数据采集：通过设备连接采集试验数据
- 设备使用登记：记录设备使用情况
- 数据上传管理：管理离线数据的上传

### 3. 模板管理
- 模板列表：查看可用的试验模板
- 模板详情：查看模板的详细信息

### 4. 系统设置
- 网络设置：配置网络连接
- 更新管理：管理应用更新
- 账户管理：管理用户账户信息
- 系统日志：查看系统操作日志

### 5. 报告管理
- 报告生成：生成试验报告
- 报告提交：提交试验报告
- PDF报告：生成PDF格式的报告

## 技术特点

1. **离线工作能力**：支持离线模式，确保在网络不稳定环境下仍能正常工作
2. **设备集成**：支持与试验设备的连接和数据采集
3. **模块化设计**：采用模块化设计，便于维护和扩展
4. **类型安全**：使用TypeScript确保类型安全
5. **响应式布局**：适配不同尺寸的平板设备
6. **数据同步**：支持离线数据与服务器的同步

## 应用场景

该软件主要应用于以下场景：

1. **实验室试验**：在实验室环境中进行试验数据的采集和管理
2. **现场检测**：在现场环境中进行检测数据的采集和管理
3. **数据上报**：将采集的数据上报到服务器进行统一管理
4. **报告生成**：根据采集的数据生成试验报告
